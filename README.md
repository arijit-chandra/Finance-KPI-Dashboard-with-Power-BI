# 📊 Finance KPI Dashboard with Power BI

An end-to-end **Finance KPI Dashboard** built using **Power BI**, covering data preparation, modeling, DAX calculations, and executive-level dashboard design.

This project demonstrates advanced Power BI capabilities including multi-fact data modeling, YTD calculations, KPI benchmarking, emoji-enhanced variance indicators, smart narrative insights, and executive-ready reporting.

---

## 🚀 Project Overview

This dashboard provides actionable financial insights through dynamic KPI tracking and interactive visuals.

### 🔹 Core Development Stages

1. **Data Preparation**

   * Connecting to raw data sources
   * Data cleaning & transformation using Power Query
   * Using **Unpivot** for reshaping financial data

2. **Data Modelling**

   * Building a semantic model
   * Creating relationships between dimension and multiple fact tables
   * Star schema design best practices

3. **DAX Calculations**

   * YTD (Year-to-Date) calculations
   * Variance analysis
   * KPI logic with conditional formatting
   * FILTER function usage
   * Emoji-based performance indicators

4. **Dashboard Design**

   * KPI Cards with reference labels
   * Win/Loss analysis chart
   * Monthly trend analysis
   * Salesperson performance matrix with sparklines
   * Smart Narrative (AI-generated insights)
   * Custom data labels
   * Executive-level formatting & themes
